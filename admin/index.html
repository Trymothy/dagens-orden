Hvordan opprette token?
  1. Gå til github.com → Settings → Developer settings → Personal access tokens → Fine-grained tokens
  2. Klikk Generate new token
  3. Under Repository access: velg kun dette repoet
  4. Under Permissions → Contents: sett til Read and write
  5. Kopier tokenet og lim det inn over
